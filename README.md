# Human AI Screening Tasks 

## ISSR 3 : Trust Calibration in AI-Assisted Decision Making
This project consists of a minimal React frontend and a FastAPI backend designed to deliver an AI recommendation task, log participant behavior, and record latencies.

## ISSR 4 : FOA Ingestion + Semantic Tagging Pipeline
Ingests an **NSF Funding Opportunity Announcement (FOA)** from a public URL, extracts structured fields using a layered extraction strategy, applies **spaCy-powered rule-based semantic tags**, and exports the result as **JSON** and **CSV**.
